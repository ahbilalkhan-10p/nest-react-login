Nest + React Auth Form / Boilerplate 

Project structure:

- client -> React
- server -> Nest

Project Setup

**Server:**

`cd  server`

`npm i`

then go back to main directory

`cd ..`

`docker-compose up --build`

**Client:**

`cd client`

`npm i` 

`npm start`


Note: only the server and db are on docker for now
